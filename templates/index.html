📘 תיעוד ה-API של התנ"ך עם גימטריה

להלן נקודות הקצה (endpoints) הזמינות:

נתיב שיטה תיאור דוגמה
/ GET דף הבית או הודעת JSON /
/tanakh/<book> GET הצגת כל הפרקים בספר /tanakh/Genesis
/tanakh/<book>/<chapter> GET פסוקים + גימטריה בפרק מסוים /tanakh/Genesis/1
/tanakh/<book>/<chapter>/<verse> GET פסוק בודד עם גימטריה /tanakh/Genesis/1/1
/tanakh/<book>/<chapter>/page/<page> GET עמוד של 10 פסוקים מתוך פרק /tanakh/Genesis/1/page/1
/tanakh/<book>/last_chapter GET הפרק האחרון הזמין בספר /tanakh/Genesis/last_chapter
/gematria POST חישוב גימטריה לטקסט חופשי (JSON בלבד) 
{
  "text": "בראשית ברא אלוהים"
}

💡 להציג JSON בדפדפן? השתמש בתוסף ModHeader והוסף כותרת: Accept: application/json
