API לגימטריה של תנ"ך - תיעוד

ברוכים הבאים ל-API שמחשב את גימטריית הפסוקים בתנ"ך.

כתובות ה-API הזמינות:

/tanakh/<ספר>

מחזיר את כל הפרקים והפסוקים של ספר מסוים.

/tanakh/<ספר>/<פרק>

מחזיר את פרק ספציפי עם כל הפסוקים שלו.

/tanakh/<ספר>/<פרק>/page/<מספר עמוד>

מחזיר חלק מהפסוקים של פרק, לפי מספר העמוד (עד 10 פסוקים בעמוד).

/tanakh/<ספר>/last_chapter

מחזיר את מספר הפרק האחרון בספר.

/tanakh/<ספר>/<פרק>/<פסוק>

מחזיר פסוק ספציפי מתוך פרק בספר.

/gematria (POST)

מחשב את גימטריית הטקסט שנשלח בבקשה.

דוגמאות לשימוש:

קבלת כל ספר בראשית:

GET /tanakh/בראשית

קבלת הפרק הראשון בספר בראשית:

GET /tanakh/בראשית/1

קבלת עמוד שני מפרק ראשון בספר שמות:

GET /tanakh/שמות/1/page/2

קבלת מספר הפרק האחרון בספר תהילים:

GET /tanakh/תהילים/last_chapter

קבלת פסוק 3 מפרק 5 בספר שמות:

GET /tanakh/שמות/5/3

חישוב גימטריה לטקסט "בראשית":

POST /gematria תוכן גוף (JSON): { "text": "בראשית" }
